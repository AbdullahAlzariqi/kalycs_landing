FAQ

Frequently Asked Questions

Quick answers about Kalycs, AI features, and privacy.

Is AI included in Kalycs?

Yes. Kalycs includes AI features like semantic search, natural language Q&A, intelligent file categorization, and smart summaries. You can find answers across your files, not just keywords.

Do I need an OpenAI API key to use Kalycs?

OpenAI API key is required. Add your key in Settings to enable AI features. Without a key, non‑AI features like local search still work.

Which AI models (LLMs) does Kalycs support?

Only OpenAI (GPT) is supported. Other providers are not supported.

Can I use Kalycs without AI?

Yes. Kalycs includes a local search engine (Bleve) that works entirely offline. You can organize, search, and manage files without AI. AI features like summaries and tagging are optional.

How does Kalycs handle my data?

Your data stays under your control. All files and searches run on your device. When you use AI features, requests are sent to OpenAI using your API key; we never share your content with third parties beyond that processing.

How are my API keys stored?

API keys are stored securely in your system's keychain, not transmitted to Kalycs servers. This ensures your credentials remain private and under your control.

Does Kalycs send my files to AI providers?

When using AI features, relevant file content may be sent to your configured AI providers for processing (summarization, chat, etc.). This follows the same privacy policies as using these AI services directly.

Note:AI capabilities (semantic search, chat with files, smart tagging) as a feature requiring OpenAI API key.
