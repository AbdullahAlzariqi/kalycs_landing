Product Roadmap

Potential user outcomes we’re exploring. No dates, not ordered, and subject to change based on feedback.

Disclaimer: These are potential features under exploration. No dates or quarters, and not an ordered plan. We’ll iterate based on feedback and feasibility.
1

See the exact answer in context

“As a user, I can see the exact snippet with surrounding context highlighted so I instantly trust the result.”

2

Chat with any file

“I can ask questions about a file and get grounded answers without hunting through pages.”

3

Continue my AI conversations

“When I come back, my previous AI chats are there so I can pick up where I left off.”

Smart drafting from my knowledge

“I can generate emails/reports by combining info from multiple files/projects in one go.”

“My files get auto-tagged with dates/authors and other metadata so filtering is quick.”

Create rules in plain language

“I can write simple, natural-language rules to auto-organize files—no regex needed.”

Drop files straight into the right project

“I can upload directly into a project so things are organized from the first moment.”

8

No more duplicates

“The app detects duplicate content so my workspace stays clean and trustworthy.”

9

Discover similar files while working

“While I’m drafting or exploring, the app suggests related files I should look at.”

10

See & recover previous versions

“I can view and restore earlier versions of a file when something goes wrong.”
