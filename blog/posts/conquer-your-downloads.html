Downloads get messy. In Kalycs, you can auto‑organize that folder with a few simple rules — or let suggestions help you.

Set up in minutes

  1. Open Kalycs and go to Settings → Organization.
  2. Enable “Watch Downloads”.
  3. Add a couple of rules, e.g. “PDF → /Documents/Receipts”.
  4. Optionally enable suggestions to review before applying.
  5. Relax — new files file themselves where they belong.
  6. Adjust rules as your needs evolve.

Tips

  • Start broad: file type and size catch most clutter.
  • Refine over time with keywords or date ranges.
  • Keep it simple — fewer, clearer rules beat many complex ones.
